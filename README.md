# koa2-test
Project showing how to use [koa 2](https://github.com/koajs/koa) and [objection](https://github.com/Vincit/objection.js/) for ORM  

## Install  
yarn install  
yarn global add knex  

## Migrations & Seeding
knex migrate:latest  
knex seed:run --env development
